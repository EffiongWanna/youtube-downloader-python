# YouTube Downloader (Python)

A command-line YouTube downloader built with Python using the pytube library.
This project allows users to download YouTube videos in different qualities
or extract audio only.

## Features
- Download highest quality video
- Download lowest quality video
- Download audio-only streams
- Error handling for invalid links and network issues

## Concepts Used
- Python input/output
- Conditional statements
- Exception handling (try/except)
- External libraries (pytube)

## How to Run
1. Install Python 3.x
2. Install dependencies:
   pip install pytube
3. Run the program:
   python main.py
